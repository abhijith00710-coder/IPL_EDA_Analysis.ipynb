# IPL Data Analysis (2008–2024): Exploratory Data Analysis Using Python

## Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on the Indian Premier League (IPL) dataset from 2008–2024 using Python. The main objective is to understand the importance of data preprocessing and how handling missing values affects the quality, reliability, and interpretation of analytical results.

Real-world datasets are rarely clean. Through this project, raw IPL data was transformed into meaningful insights using data cleaning, preprocessing, and visualization techniques.

---

## Problem Statement

Raw datasets often contain missing values, inconsistent entries, duplicate records, and incorrect data types that can affect analysis quality. The goal of this project is to:

* Identify and handle missing values
* Clean and preprocess raw IPL data
* Transform data into an analyzable format
* Explore patterns and trends using visual analysis
* Generate meaningful insights from the dataset

---

## Dataset Information

Dataset: IPL Match Dataset (2008–2024)

The dataset contains information related to:

* Match details
* Teams and seasons
* Players
* Match outcomes
* Venues
* Toss decisions
* Scores and statistics

## Project Workflow

### 1. Data Collection
